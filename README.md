# Christoffel's Menu Manager

A React Native mobile application for chefs to build, manage, and showcase their restaurant menu. The app includes a **preloaded starter menu**, allows **custom dish creation**, and lets users **filter by course** ‚Äî all in a clean, grey-themed interface.

- **Preloaded Sample Menu**:
  - ü•ó **Starters**: Garlic Butter Shrimp
  - ü•© **Mains**: Grilled Ribeye Steak
  - üç∞ **Dessert**: Chocolate Cake
- **Add custom dishes** with:
  - Dish name
  - Description
  - Course (Starters, Mains, Dessert)
  - Price (in Rands)
- **Remove any item** (sample or custom) from your menu
- **View full menu** with images, descriptions, and prices
- **Filter dishes** by course (Starters / Mains / Dessert)
- **Real-time stats** on Home screen:
  - Total number of menu items
  - Average price per course
- **Clean grey UI** ‚Äì professional and easy on the eyes
- **Fully responsive** ‚Äì works on all screen sizes
- **Expo-compatible** ‚Äî runs instantly on Android & iOS via Expo Go
